Python

Python Tkinter Pillow

A Python GUI program that composites a watermark text to a desired image. The code for this program utilizes the tkinter module for the GUI part and Pillow module for the watermarking part.

When the user opens the program, the user will be prompted to input the watermark text and set the font-name, font-size, and opacity. After the user clicks the “Add Text” button, the program will open the image file as a base from the designated folder. The program will then composite another image with the watermark text that the user has set earlier over the base image. The final image with the watermark text is then saved in the designated folder.

Typing Speed Test

Python Tkinter

A Python GUI program that tests typing speed. When the user opens the program, the user is first greeted with the main title but the test doesn’t begin until the user presses the start button. After the start button is pressed, the timer begins counting down from 60 seconds and the user will need to type the words as they are displayed on the screen into the entry widget as quickly as they can. After the user is done typing the word, they would need to press the space bar to get the next word to type. As the game progress, the words that the user typed are saved for checking at the end of the test. Once the countdown ends, the program would compare the words typed by the user with the words generated to calculate the score. The score is then displayed on the GUI for the user.
