Todo App
Python Flask HTML CSS Bootstrap

A web application for managing tasks and projects. It is built with Flask framework for the backend while Bootstrap framework is used for the frontend. The backend is connected to a database to store user data. Users will need to be registered before using the app. Once registered and logged in, users can create projects and add tasks within projects that they created. Users can also edit and delete their tasks when needed. Users can view their assigned tasks in the My Task page and mark on the tasks that they have completed. Besides creating tasks for themselves, users can also create tasks and assign them to other users on projects that they want to collaborate with others. Users can also comment on tasks to facilitate collaboration.

Weather App
HTML CSS Javascript React Axios

A web application for checking weather at any location, built using React framework. The weather data is obtained from OpenWeather by making API calls using the Axios library. React useState hooks are used to track changes in the weather data. The app takes user input coordinates and then displays the current weather and 5-day weather forecast for that particular location. User can also view the detailed forecast (at 3-hour intervals) for each of the 5 days. The detailed forecast only displays data for 1 day at a time. The user can switch between days by clicking the tabs for each of the 5 days.

Typing Speed Test
Python Tkinter

A Python GUI program that tests typing speed. When the user opens the program, the user is first greeted with the main title but the test doesn’t begin until the user presses the start button. After the start button is pressed, the timer begins counting down from 60 seconds and the user will need to type the words as they are displayed on the screen into the entry widget as quickly as they can. After the user is done typing the word, they would need to press the space bar to get the next word to type. As the game progress, the words that the user typed are saved for checking at the end of the test. Once the countdown ends, the program would compare the words typed by the user with the words generated to calculate the score. The score is then displayed on the GUI for the user.

Image Watermarking Desktop App
Python Tkinter Pillow

A Python GUI program that composites a watermark text to a desired image. The code for this program utilizes the tkinter module for the GUI part and Pillow module for the watermarking part. When the user opens the program, the user will be prompted to input the watermark text and set the font-name, font-size, and opacity. After the user clicks the “Add Text” button, the program will open the image file as a base from the designated folder. The program will then composite another image with the watermark text that the user has set earlier over the base image. The final image with the watermark text is then saved in the designated folder.
